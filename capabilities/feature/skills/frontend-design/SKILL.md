# Frontend Design Skill

Use this skill for user-facing UI work.

## Kody Dashboard Adaptation

- Choose an intentional visual direction, but keep it suitable for a work dashboard.
- Avoid landing-page composition, oversized heroes, decorative cards, and one-note palettes.
- Make information easy to scan, compare, and act on.
- Use the local `design-system` skill first when it is available.
- Prefer strong hierarchy, spacing, typography, and interaction polish over decoration.
- Every screen must work on mobile and desktop.
