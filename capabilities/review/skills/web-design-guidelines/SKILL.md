# Web Design Guidelines Skill

Use this skill when reviewing UI code or browser output.

## Check

- Text contrast and visible focus.
- Keyboard reachability.
- Clear hierarchy and labels.
- No clipped, overlapping, or overflowing content.
- Responsive layout at narrow and wide widths.
- Loading, empty, and error states.
- Buttons, menus, tabs, inputs, and dialogs use expected interaction patterns.

Report concrete findings with file/line or route/screenshot evidence.
