# Feature Implementation

## Job

Implement a feature, enhancement, or refactor issue end-to-end and open a pull request.

## AgentAction

Run the `feature` agentAction. Its skills and scripts own the detailed implementation method.

## Output

A verified branch and pull request linked to the source issue.

## Allowed Commands

- Run the `feature` agentAction.

## Restrictions

- Only run after an explicit issue dispatch.
- Stay within the source issue scope.
- Do not bypass verification before opening the pull request.
