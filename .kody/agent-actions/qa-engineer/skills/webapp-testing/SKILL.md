# Webapp Testing Skill

Use this skill for browser verification.

## Workflow

1. Confirm the target URL and auth state.
2. Start or locate the dev server only when needed.
3. Inspect the page before acting.
4. Exercise the smallest path that proves the behavior.
5. Check loading, empty, error, mobile, and keyboard states when relevant.
6. Capture screenshots only when they prove a finding or a fix.
7. Report what was verified and what could not be verified.
