# PR Branch Sync

## Job

Bring a pull request branch up to date with its base branch.

## AgentAction

Run the `sync` agentAction. The engine owns the implementation details.

## Output

An updated pull request branch.

## Allowed Commands

- Run the `sync` agentAction.

## Restrictions

- Only sync the target pull request branch.
- Do not add unrelated code changes.
- Do not merge the pull request from this agentResponsibility.
