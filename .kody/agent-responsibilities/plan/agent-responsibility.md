# Issue Planning

## Job

Research an issue and produce an implementation plan without editing code.

## AgentAction

Run the `plan` agentAction. Its planning skills own the detailed method.

## Output

A clear plan attached to the issue or returned by the engine.

## Allowed Commands

- Run the `plan` agentAction.

## Restrictions

- Do not edit files.
- Do not open a pull request.
- Stop after the plan unless another agentResponsibility is explicitly dispatched.
