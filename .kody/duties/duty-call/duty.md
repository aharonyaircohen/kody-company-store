# Duty Call

## Job

Propose one high-ROI missing duty that the system does not already have.

## Executable

Run the `duty-call` executable. Its skill owns the detailed method and runtime state handling.

## Output

A proposal issue for operator approval.

## Allowed Commands

- Run the `duty-call` executable.

## Restrictions

- One proposal per tick.
- Do not create the duty directly.
- Never re-propose rejected ideas.
- Respect dismissed ideas until their cooling-off window expires.
