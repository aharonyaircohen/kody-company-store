# Issue Research

## Job

Investigate an issue and report relevant facts, options, and risks.

## Executable

Run the `research` executable. Its research skill owns the detailed method.

## Output

A concise research result for the target issue.

## Allowed Commands

- Run the `research` executable.

## Restrictions

- Do not edit files.
- Separate verified facts from recommendations.
- Do not dispatch implementation work from this duty.
